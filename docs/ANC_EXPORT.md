# ANC Export

Use ANC export when you need an Austrian NeuroCloud-oriented delivery package.

If you are looking for the broad output workflow (derivatives, anonymized bundles, openMINDS), start with [ANALYSIS_OUTPUT.md](ANALYSIS_OUTPUT.md).

## What ANC export does

- creates a dedicated export folder (separate from your working dataset)
- preserves the main dataset structure as your source of truth
- supports submission preparation workflows that require ANC-specific packaging

## Minimal workflow

1. Open your project in PRISM Studio.
2. Finish validation first.
3. Go to the project export area and run **ANC export**.
4. Review the generated `_anc_export` output folder before submission.

## Practical checks before submission

- confirm required metadata fields are present
- verify participant identifiers match your intended submission policy
- ensure you are exporting the correct project state (latest validated version)
