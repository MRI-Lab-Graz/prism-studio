# Create New Survey Workbook

Canonical location for the survey template authoring workbook used to create new survey JSON templates.

## Primary workbook
- survey_import_template.xlsx

## Notes
- This workbook is intended to be software-agnostic (Excel, LibreOffice, Apple Numbers).
- Use Items.ApplicableVersions per item row.
- For multiple variants in one item, enter plain text lists (e.g. `10-likert;7-likert`).
- Variants sheet defines available VariantID values and optional per-item variant scale overrides.
