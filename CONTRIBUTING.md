# Contributing to PRISM

Thank you for considering contributing to PRISM. By contributing, you agree that your
contributions will be licensed under the GNU Affero General Public License v3.0 (AGPL-3.0).

Please follow the repository's contribution guidelines:

- Fork the repository and create feature branches.
- Run tests: `./run_tests.sh`.
- Open a pull request describing your changes.

If you are an organization or otherwise cannot agree to this, please contact us.
