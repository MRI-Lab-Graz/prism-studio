# ‚úÖ Workshop Screenshot System - Summary

The PRISM workshop uses **Heroshot** - an interactive visual tool for screenshot management.

## üìù Important: Heroshot is Interactive

**Heroshot v0.13+ is NOT a config-file tool.**  It's an **interactive visual editor** that opens a browser where you:

1. Navigate to pages
2. Click to select elements
3. Add styling/annotations visually
4. Save screenshot definitions

The `config.json` file is **auto-generated** by Heroshot - you don't write it manually.

## üìö Files Created

### `.heroshot/` Directory

1. **`config.json`**
   - Auto-generated by Heroshot's visual editor
   - DO NOT edit manually
   - Stores screenshot definitions after visual configuration

2. **`HEROSHOT_SETUP.md`**
   - Complete setup guide for interactive workflow
   - Explains visual editor usage

3. **`README.md`**
   - Quick reference for .heroshot directory

4. **`capture-workshop-screenshots.sh/.bat`** 
   - Helper scripts (deprecated - not compatible with Heroshot v0.13+)
   - Use `npx heroshot` instead
