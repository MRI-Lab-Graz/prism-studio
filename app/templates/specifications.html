The Core Concept

PRISM is a hybrid dataset validation tool designed specifically for psychological experiments. It builds upon the BIDS (Brain Imaging Data Structure) standard while adding stricter metadata requirements and support for non-imaging modalities.

BIDS Compatible

PRISM datasets remain valid BIDS datasets. Any BIDS-compatible tool (like fMRIPrep) will still work perfectly with your PRISM-structured data.

PRISM Enhanced

PRISM adds specific schemas for surveys, biometrics, and physiological data that go beyond the standard BIDS requirements to ensure high-quality metadata.

Standardized Structure

Consistent naming conventions (sub-ID, ses-ID, task-ID) across all modalities.

Rich Metadata

Every data file must have a JSON sidecar containing technical and descriptive metadata.

Validation First

Automated checks ensure your dataset is complete and follows the specification before analysis.

Detailed Schema Documentation

For a deep dive into the specific JSON schemas and modality requirements, please visit our dedicated documentation on ReadTheDocs.
