# Workshop Structure (Quick Reference)

## Flow used in the 2-hour session

1. `exercise_0_project_setup/INSTRUCTIONS.md` — download/setup + create project
2. `exercise_5_participant_mapping/INSTRUCTIONS.md` — import participant data
3. `exercise_1_raw_data/INSTRUCTIONS.md` — import survey data
4. `exercise_2_hunting_errors/INSTRUCTIONS.md` — error hunting
5. `exercise_3_using_recipes/INSTRUCTIONS.md` — scoring/export
6. `exercise_4_templates/INSTRUCTIONS.md` — templates

- `exercise_0_project_setup/`
